# Meu Portfólio

Portfólio pessoal desenvolvido com Next.js, React e Tailwind CSS. Apresenta projetos, experiência profissional, habilidades e informações de contato.

## Tecnologias

- **Next.js** - Framework React com SSR e SSG
- **React** - Biblioteca UI
- **Tailwind CSS** - Estilização
- **TypeScript** - Tipagem estática
- **shadcn/ui** - Componentes reutilizáveis

## Como usar

### Instalação

```bash
pnpm install
```

### Desenvolvimento

```bash
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000)

### Build para produção

```bash
pnpm build
pnpm start
```

## Estrutura

- `app/` - Páginas principais (Next.js App Router)
- `components/` - Componentes reutilizáveis
- `lib/` - Utilitários
- `public/` - Arquivos estáticos
- `styles/` - Estilos globais
